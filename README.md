# AutoCalypsoBTS

A simple GUI for CalypsoBTS for DragonOS 

## Install :
`git clone https://github.com/jhonnybonny/AutoCalypsoBTS.git && cd AutoCalypsoBTS && sudo bash install.sh`
## GUI start :
`cd autocalypsobts && sudo python3 autobts.py`
